案例分析与Github Fork

名称 内容
案例 哈佛大学计科系《机器学习》课程网站
工作1：分析与学习
  1. 访问网站，基于你已经学过的知识，想象它是如何实现的
  2. 在浏览器中查看网页源代码，看是否和你想象的一致。记录下你觉得新颖的内容
  3. 该网页源代码的第18行左右，用到了jquery这个东东，百度一下，这是什么？
  4. 该网页源代码的第29行左右，用到了bootstrap这个东东，百度一下，这是什么？
  5. 该网页源代码的第78行左右，用到了dl，dt，dd这些HTML标签，东东，百度一下，这是什么？它们和我们学过的Table有什么区别？
  6. 该网页源代码的第148行左右，用到了{% for xx in xx %}，你能猜出它的作用吗？
参考文献
  1. jquery官方网站
  2. bootstrap官方网站
工作2：复制和修改
  1. 访问该网站的Github代码，观察它的目录和文件组织方式
  2. 用你的账号登陆Github，参考 Fork 一个示例仓库，复制该网站到你的Github账号下
  3. 设置复制过来的网站，在浏览器访问它。是的，现在它变成你的了
  4. 在你的电脑上，打开终端窗口，Git克隆该网站到你的电脑里
  5. 用Sublime Text修改网站标题为《实务学堂写作课》，在自己电脑上打开看效果
  6. 用git commit，push，将修改好的网站上传到Github上，请欧阳老师审阅

Sublime Text安装与使用

名称 内容
安装 官方网站下载安装
常见命令
  1. Ctrl + D：选择当前光标所在的词并高亮该词所有出现的位置，再次Ctrl + D选择该词出现的下一个位置，在多重选词的过程中，使用Ctrl + K进行跳过，使用Ctrl + U进行回退，使用Esc退出多重编辑
  2. Ctrl + J：把当前选中区域合并为一行
  3. Ctrl + F2：添加/移除书签
  4. F2：跳到下一个书签
  5. Ctrl + Shift + L: 对一片区域的所有行进行同时编辑
  6. Ctrl + Cmd + G: 全选选中的文本

B站投稿和视频嵌入方法

名称 内容
介绍
  1. 登录B站后，访问 B站创作中心 - 视频投稿投稿页面，上传视频。
  2. 上传视频时，不要怕麻烦，要详细给出视频的各种标签、介绍哦，这样B站才会知道如何把你的视频推荐给想寻找这些视频的人
  3. 记得在视频的介绍中给出你的Github网站链接哦，这样就会很多人来访问你的网站！
  4. 耐心等待审核通过（一小时左右）
  5. 审核通过后，参考 用iframe/embed在网页上嵌入视频/音乐组件 文中介绍方法，获得你的视频的iframe代码，粘贴到我们的Github网站代码中（可以参考下面的示例）
  6. 把文件上传到Github上，然后用浏览器访问你的网站，就可以看到你的B站视频喽！（如果是本机上打开网页，可能看不到。没关系。上传到网站上，就可以看到了。
